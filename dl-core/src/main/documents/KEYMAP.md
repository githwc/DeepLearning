[搜索]
    1.当前文件搜索文本内容      ctrl + F
    2.全局搜索文本内容         ctrl + shift + F
    3.按名字搜索类            ctrl + N
    4.按名字搜索所有文件        ctrl + shift + N
    5.controller到impl      ctrl + Alt + 左键
    6.查找类或方法在哪被调用    Alt + F7
    
[编辑]
    1.最大化编辑区            ctrl + shift + F12
    2.文件重命名              ctrl + F6
    3.对当前文本内容进行替换    ctrl + R
    4.对全局文本内容进行替换    ctrl + shift + R
    5.删除当前行或选中行        ctrl + Y
    6.移动光标向上/向下         Alt + Shift + 方向

[快捷]
